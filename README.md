ğŸ“„ Abstract

The project focuses on ensuring child and women safety using Received Signal Strength (RSS) instead of GPS to reduce cost and increase indoor tracking efficiency. The proposed system monitors movement through RSS-based localization and triggers alerts when critical conditions are detected. Wearable sensor modules like MEMS help detect potential danger signals, and notifications are sent to parents or guardians.
